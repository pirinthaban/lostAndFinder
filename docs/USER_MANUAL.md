## Posting a Lost Item

### Step-by-Step Guide

1. **Tap the "+" button** at the bottom center
2. Select **"I Lost Something"**
3. Fill in the details:

   **Basic Information:**
   - Category: Select from NIC, Phone, Wallet, etc.
   - Title: Brief description (e.g., "Black Leather Wallet")
   - Description: Detailed info (brand, color, unique features)
   
   **Photos:**
   - Upload 1-5 clear photos
   - System will auto-blur sensitive details
   
   **Location:**
   - Allow GPS access
   - Confirm location on map
   - Adjust pin if needed
   - Set search radius (500m - 50km)
   
   **Date & Time:**
   - When did you lose it?
   
   **Urgency:**
   - Low: Non-critical items
   - Medium: Important items
   - High: Very important (NIC, Passport)
   - Emergency: Urgent medical items

4. **Review & Post**
5. You'll receive match notifications if similar items are found

### Best Practices

‚úÖ **DO:**
- Upload clear, well-lit photos
- Include unique identifying features
- Be specific in description
- Update if you find the item

‚ùå **DON'T:**
- Share sensitive personal information
- Post fake items
- Use abusive language

## Understanding AI Matches

### What is AI Matching?

Our smart system automatically finds potential matches between lost and found items using:
- **Image Similarity** (40%): Visual comparison
- **Text Match** (30%): Description keywords
- **Location** (20%): Proximity
- **Time** (10%): When it was lost/found

## Claiming an Item

### Claim Process

1. **Initiate Claim**
   - Find the item
   - Tap "Claim This Item"
   
2. **Provide Proof**
   - Upload photos you have of the item
   - Answer security questions
   - Provide any receipts/documents
   
3. **Owner Review**
   - Item owner reviews your claim
   - May ask additional questions
   - Reviews your profile & reputation
   
4. **Verification**
   - For high-value items: Police verification
   - For NIC/Passport: Extra security checks
   
5. **Arrange Handover**
   - Use in-app chat
   - Agree on public meeting place
   - Get 6-digit handover code
   
6. **Complete Handover**
   - Meet in person
   - Exchange handover code
   - Confirm in app
   - Both parties confirm completion

### Safety During Handover

‚úÖ **Safe Meeting Places:**
- Police stations
- Shopping malls
- Public libraries
- University campus
- Bank branches

‚ùå **Avoid:**
- Private homes
- Dark/isolated areas
- Late night meetings

### Handover Code

- 6-digit code ensures authenticity
- Only share with the other party
- Enter code in app to complete
- Item status changes to "Closed"

## Troubleshooting

### Common Issues

**1. Can't receive OTP**
- Check phone number format
- Check SMS inbox
- Ensure good signal
- Wait 2 minutes and retry
- Contact support if persistent

**2. Upload failing**
- Check internet connection
- Ensure image <5MB
- Try compressing photo
- Check storage space
- Restart app

**3. App crashing**
- Update to latest version
- Clear app cache
- Restart device
- Reinstall app
- Report bug to support

**4. Not seeing nearby items**
- Enable location permissions
- Check GPS is on
- Increase search radius
- Try different filters
- Refresh page

**5. Matches not appearing**
- Wait 5-10 seconds after posting
- Check item has clear photos
- Ensure description is detailed
- Try editing item to trigger re-match

## Frequently Asked Questions

**Q: Is the app free?**
A: Yes! Free tier includes 3 posts/month. Premium unlocks unlimited posts and features.

**Q: How accurate is AI matching?**
A: 75-85% accuracy for high-quality photos. Always verify manually.

**Q: What if I find my item elsewhere?**
A: Mark item as "Found" in app to close the case.

**Q: Can I post on behalf of someone?**
A: Yes, but mention it in description.

**Q: How long do items stay active?**
A: Lost items: 90 days. Found items: 60 days. Can be extended.

**Q: Is my data safe?**
A: Yes. We use bank-level encryption and comply with data protection laws.

**Q: Can I use app offline?**
A: Yes! Post items offline; they'll sync when online.

**Q: What if someone claims my found item falsely?**
A: You review all claims. Ask for proof. Police can verify high-value items.
