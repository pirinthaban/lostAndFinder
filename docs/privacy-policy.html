Introduction

Welcome to FindBack. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our mobile application and services.

Please read this Privacy Policy carefully. By using FindBack, you agree to the collection and use of information in accordance with this policy.

1. Information We Collect

1.1 Personal Information You Provide

  • Account Information: Phone number (required), email address (optional), display name, profile photo
  • Item Information: Photos of lost/found items, descriptions, location data, timestamps
  • Communication Data: In-app messages (encrypted), claim submissions, feedback
  • Verification Data: Ownership proof documents, security question answers

1.2 Information Collected Automatically

  • Device Information: Device type, operating system, unique identifiers
  • Usage Data: Features accessed, interaction patterns, search queries
  • Location Data: GPS coordinates (approximate), IP address

2. How We Use Your Information

  • To provide and maintain the Service
  • Match lost and found items using AI
  • Enable communication between users
  • Process ownership claims
  • Prevent fraud and ensure security
  • Improve app performance and features
  • Send service-related notifications
  • Comply with legal obligations

3. Data Protection & Security

Encryption

  • Data in Transit: TLS 1.3 / HTTPS
  • Data at Rest: Firebase encryption
  • Chat Messages: End-to-end encryption (AES-256)

Privacy Features

  • Auto-blur sensitive data (NIC numbers, faces)
  • Location fuzzing (approximate area only)
  • Phone number masking
  • No exact GPS coordinates shared publicly
